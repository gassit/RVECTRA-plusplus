## Расчёт потери напряжения

**Два метода:**

### 1. Точный метод (есть R₀/X₀ из справочника)
```
ΔU% = √3 × I × (R×cosφ + X×sinφ) / U × 100
```

### 2. Упрощённый метод (только ρ и S)
```
ΔU% = (ρ × L × P × 100) / (U² × S × cosφ)
```

**Автовыбор в `calculateVoltageDropAuto()`:**
- Если `r0OhmPerKm > 0` → точный метод
- Иначе → упрощённый метод
